# gmpy安装

前几天把wsl搞炸了，重装一遍

装gmpy的时候发现了一种比较方便的方法

```
apt-get install libgmp-dev
apt-get install libmpfr-dev
apt-get install libmpc-dev
pip install gmpy
pip install gmpy2
```

我第一次装的时候折腾半天真是蠢
