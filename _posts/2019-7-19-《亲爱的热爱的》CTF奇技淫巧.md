# 《亲爱的热爱的》CTF奇技淫巧

* ### nmap防御

  * #### nmap安装

    ```
    apt-get install nmap
    ```

  * #### 修复漏洞与安全加固

    ```
    mv ~/dev/null
    grep root /etc/nmap
    nmap -sT -A localhost
    ```
