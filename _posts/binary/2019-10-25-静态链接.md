# 静态链接

## 静态库

命名规则lib(.*).a



## 静态链接

在链接阶段，将目标文件与引用的库一起打包到可执行文件中

有无函数库对程序运行没有影响，移植方便，但是目标文件和引用的库合成为一个可执行文件，浪费空间



## 样例

```c
#include <stdio.h>
int	main(int argc, char const *argv[])
{
	puts("hello world");
	return 0;
}
```

```
gcc hello.c -static -o hellos
file hellos

```
